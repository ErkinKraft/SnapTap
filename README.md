# SnapTap

Режим Snap Tap (last-input-wins) для WASD: при одновременном зажатии противоположных направлений приоритет у последней нажатой клавиши. Если её отпустить, а противоположная ещё зажата - движение снова идёт в ту сторону.

## Установка

1. Скачайте последний релиз **или**:
   ```
   pip install keyboard
   python snaptap1.py
   ```
2. При старте программа сама запросит права администратора.

## Использование

1. Запустите программу - откроется тёмный GUI.
2. Нажмите **Активировать**.
3. Остановка: кнопка **Деактивировать**, **F9** или **Esc**.
4. **F10** - окно «О программе».

Раскладка клавиатуры (ENG/RU) не важна - используются физические клавиши WASD.
Клавиши WASD на экране подсвечиваются при активных осях.
